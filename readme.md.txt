# Personal Expense Tracker - README

## 📌 Project Overview
A simple **Personal Expense Tracker** built using **Flask** for tracking your daily expenses, viewing summaries, and maintaining persistent data using JSON files. It helps you visualize spending by category and date in a clean, user-friendly UI.

---

## 🚀 Features
✅ Add daily expenses with amount, category, and date.  
✅ View all expenses in a clean, card-based interface.  
✅ View total spending and category-wise breakdown in a summary view.  
✅ Persistent storage using `expenses.json`.  
✅ Mobile-friendly, clean UI with Bootstrap styling.  
✅ Ready for deployment on Render, PythonAnywhere, or your local server.

---

## 🗂️ Folder Structure
```
project/
│
├── app.py               # Flask application (backend)
├── expenses.json        # Stores your expenses (auto-generated)
└── templates/           # Contains HTML files
    ├── index.html
    ├── add.html
    └── summary.html
```

---

## ⚙️ Requirements
- Python 3.x
- Flask

Install dependencies using:
```bash
pip install flask
```

---

## 🛠️ Running the Project
1️⃣ Clone or download this repository.  
2️⃣ Ensure `app.py` and `templates/` are in the same directory.  
3️⃣ Run:
```bash
python app.py
```
4️⃣ Open your browser and visit:
```
http://127.0.0.1:5000/
```

---

## 📈 Usage
- **Add Expense:** Click '➕ Add Expense' to add your spending with amount, category, and date.
- **View Summary:** Click '📊 View Summary' to see total and category-wise analysis.
- **View All Expenses:** Home page lists all expenses in card format.

---

## ✅ Next Improvements
- Add charts for category-wise spending.
- Add user authentication for multi-user tracking.
- Export expenses to CSV or PDF.
- Dark mode toggle.
